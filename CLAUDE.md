# know-it-break-it

あたまのネジとはずしかた — Know It. Break It.

型を知って、武器にしろ。

## このリポジトリについて

AIが生成し人間が確認した静的コンテンツ＋正解のあるクイズで構成する教育サイト。

## 技術スタック

- Astro 5 + TypeScript + Tailwind CSS
- Cloudflare Pages（無料枠）
- i18n: en / ja

## 開発

```bash
npm install
npm run dev    # localhost:4321
npm run build  # ビルド確認
```
